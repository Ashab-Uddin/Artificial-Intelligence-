# 🤖 CSE 316 — Artificial Intelligence Lab

**Green University of Bangladesh | Department of CSE | Spring 2026**

---

## 📌 Course Overview
This repository contains all lab experiments, code implementations, lab reports, and project work for **CSE 316 - Artificial Intelligence Lab** at Green University of Bangladesh.

The course explores practical AI techniques through hands-on Python programming — from classic search algorithms to fuzzy logic and Bayesian networks.

---

## 🎯 Course Objectives
- Develop practical skills in Python programming including basic operations and problem-solving techniques  
- Apply various search algorithms, constraint satisfaction methods, and optimization techniques  
- Implement advanced AI concepts such as Genetic Algorithms, Bayesian Networks, Fuzzy Logic, and adversarial search strategies  

---

## 🧪 Lab Experiments

| # | Experiment | Topics Covered |
|--|-----------|--------------|
| 01 | Introduction to Python (Part 1) | Basics, syntax, data types |
| 02 | Basic Operations on Python (Part 2) | Functions, loops, data structures |
| 03 | Breadth-First Search (BFS) | Graph traversal, problem solving |
| 04 | Depth-First Search (DFS) | Recursive search, backtracking |
| 05 | Depth-Limited & Iterative Deepening Search | DLS, IDDFS |
| 06 | A* Search Algorithm | Heuristics, optimal pathfinding |
| 07 | Genetic Algorithm | Evolutionary computation |
| 08 | Constraint Satisfaction Problems (CSP) | Constraint propagation |
| 09 | N-Queens Problem (Backtracking) | Combinatorial optimization |
| 10 | Alpha-Beta Pruning | Adversarial search, game trees |
| 11 | Bayesian Networks | Probabilistic reasoning |
| 12 | Fuzzy Logic | Fuzzy sets, inference systems |
| 13 | Constraint Logic Programming (CLP) | Declarative constraint solving |
| — | Capstone Project | Real-world AI application |
